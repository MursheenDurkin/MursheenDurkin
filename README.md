# Hey, I'm Durkin 👋

Final-year Cyber Security and Networking student. I build things that scratch
an itch — network security tools, Discord bots, robot autonomy scripts, and
whatever lands on the Pi next.

---

## What I'm working on

- **SafeHaven** — Pi-based network security system: firewall, WireGuard VPN,
  IDS integration, Flask dashboard
- **Spot & Atlas** — two self-hosted Discord bots for a university esports
  society, named after Boston Dynamics robots. Spot manages the server; Atlas
  runs the competitive side. Both fully replace third-party bots, code stays mine
- **Anomaly-Based IDS** — final-year dissertation: Suricata on Pi 4 +
  Cisco Catalyst 2960 (SPAN port), anomaly detection pipeline
- **Spot SDK** — custom autonomy scripts for Boston Dynamics Spot
- **Pi 5 project portfolio** — varied personal projects across different
  domains as the hardware settles in
